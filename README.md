## Project Overview

This Power BI report helps understand elector distribution across various regions of Mumbai City.  
With the help of calculated measures, slicers, drill-downs, and KPI indicators, the dashboard enables users to explore elector trends and patterns effectively.

The objective of this dashboard is to analyze voter participation trends across Mumbai constituencies by comparing electoral roll growth, demographic turnout behavior, and party participation between 2019 and 2024 in order to identify engagement gaps and potential areas of electoral under representation.
